## Quick Start

Get chatting with Zefir in minutes. Follow these steps to launch your first conversation.

<Steps>

<Step title="Sign Up" icon="user-plus">

Create an account at `https://dashboard.example.com` to access your EV Zefir dashboard.

</Step>

<Step title="Get Your Key" icon="key">

Generate an API key from the dashboard settings. Copy `YOUR_API_KEY` securely.

<Callout kind="tip">

Store your key as an environment variable: `ZEFIR_API_KEY=YOUR_API_KEY`.

</Callout>

</Step>

<Step title="Send a Message" icon="message-circle">

Use the API to start chatting. Here's a simple example:

<CodeGroup tabs="JavaScript,Python">

````javascript
const response = await fetch('https://api.example.com/v1/chat', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${YOUR_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    message: 'Tell me a joke!',
    model: 'zefir-fun'
  })
});
const data = await response.json();
console.log(data.reply);
````

````python
import requests

headers = {
    'Authorization': f'Bearer {YOUR_API_KEY}',
    'Content-Type': 'application/json'
}
data = {
    'message': 'Tell me a joke!',
    'model': 'zefir-fun'
}
response = requests.post('https://api.example.com/v1/chat', json=data, headers=headers)
print(response.json()['reply'])
````

</CodeGroup>

</Step>

</Steps>
