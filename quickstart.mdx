## Prerequisites

Before diving in, ensure you have:

- A modern web browser like Chrome or Firefox for web access
- An email address for account verification
- Optional: The EV mobile app from App Store or Google Play for on-the-go chats

<Callout kind="info">
  No credit card required for the free tier. Upgrade later for unlimited fun conversations.
</Callout>

## Create Your EV Account

Follow these steps to set up your account on the EV platform.

<Steps>
  <Step title="Visit the Signup Page" icon="user-plus">
    Navigate to `https://app.ev.com/signup`.

    Fill in your email, create a strong password, and complete the CAPTCHA.
  </Step>
  <Step title="Verify Your Email" icon="mail">
    Check your inbox for a verification email from EV.

    Click the link to activate your account. This takes `<30` seconds.
  </Step>
  <Step title="Complete Profile" icon="settings">
    Return to `https://app.ev.com/dashboard`.

    Add your display name and avatar for personalized chats.
  </Step>
</Steps>

## Initiate Your First Fun Chat Session

Get chatting right away with these quick steps.

<Steps>
  <Step title="Start a New Conversation" icon="message-circle">
    In the chat interface, type "Hello Zefir!" or any fun prompt.

    Zefir responds with humor and wit.
  </Step>
  <Step title="Embed in Your Site (Optional)" icon="code">
    For web integration, use this snippet:

    <CodeGroup tabs="JavaScript,React">
      ````javascript
      const script = document.createElement('script');
      script.src = 'https://chat.ev.com/embed/zefir.js?key=YOUR_API_KEY';
      document.head.appendChild(script);
      ````

      ````jsx
      import { ZefirChat } from '@ev/zefir-sdk';

      function App() {
        return <ZefirChat apiKey="YOUR_API_KEY" theme="#f7cf34" />;
      }
      ````
    </CodeGroup>
  </Step>
  <Step title="End and Save Chat" icon="save">
    Type "/end" or click the close button.

    Chats auto-save to your history.
  </Step>
</Steps>
