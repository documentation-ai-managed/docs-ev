## Getting Started

To start chatting with EV:

1. Go to `https://zefirchat.com`
2. Log in with your account
3. Click the "Start Chatting" button
4. Select "ZEFIR FUN CHATBOT EV" from the bot list
5. Type your message in the input box below
6. Press Enter to send

EV will reply instantly. No special commands required. Just chat naturally.

Tip: You can adjust the chat settings anytime by clicking the settings icon.
