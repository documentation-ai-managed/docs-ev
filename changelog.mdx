## Recent Updates

Stay informed about new features, bug fixes, and improvements to EV Zefir Fun Chatbot. You receive automatic updates when available. Check this page regularly for the latest version details.

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added five new humor packs: Dad Jokes, Puns, Sci-Fi Wit, Animal Antics, and Tech Roasts. Access them via the `/humor pack-name` command.
- Introduced customizable response delays for more natural conversations. Set with `delay: 500ms` in your config.

## Improvements

- Enhanced conversation memory to retain context across 50+ exchanges.
- Optimized token usage, reducing costs by up to 20% for long sessions.

## Bug Fixes

- Fixed intermittent crashes during high-load emoji processing.
- Resolved issue where custom humor packs failed to load on restart.

</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Voice mode integration: Responds with audio clips using `voice: true` in requests.
- Multi-language support for humor packs in English, Spanish, and French.

## Bug Fixes

- Patched security vulnerability in webhook validation (CVE-2024-5678).
- Corrected parsing errors for complex user inputs with nested emojis.

## Improvements

- UI dashboard now shows real-time humor pack usage stats.

</Update>

<Update label="2024-08-10" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Core humor engine with initial packs: Classic, Sarcasm, and Riddles.
- Webhook support for integrating EV into your apps at `https://your-webhook-url.com/webhook`.

## Breaking Changes

- Updated API endpoint from `/chat` to `/zefir-chat` for better versioning.

## Bug Fixes

- Initial stability fixes for session timeouts.

</Update>
