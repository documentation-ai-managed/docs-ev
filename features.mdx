## Discovering Built-in Games and Quizzes

Zefir packs mini-games like trivia, riddles, and guess-the-word. Start with commands like "Play trivia" or "Riddle me this."

<CodeGroup tabs="Trivia,Riddle">
  ````bash
  User: Play trivia
  Zefir: Category: Science. What planet is closest to the Sun? A) Venus B) Mercury
  User: B
  Zefir: Correct! +1 point. Next: Who invented the telephone?
  ````
  ````bash
  User: Riddle me this
  Zefir: I speak without a mouth and hear without ears. I have no body, but I come alive with the wind. What am I?
  User: Echo
  Zefir: Spot on! Want another?
  ````
</CodeGroup>

## Customizing Chat Preferences

Tailor Zefir to your style. Use simple commands or integrate via API for apps.

<Expandable title="Advanced Customization" default-open="false">
  Set preferences like humor level or themes.

  ```javascript
  // Example: Embed Zefir with custom config
  const zefirConfig = {
    humorLevel: 'high',
    defaultTheme: 'pirates',
    apiKey: 'YOUR_API_KEY'
  };

  ZefirChat.init(zefirConfig);
  ```

  Adjust on-the-fly: "Set humor to medium" or "Prefer sci-fi themes."
</Expandable>

| Preference | Command Example | Effect |
|------------|-----------------|--------|
| Humor Level | "Set humor low" | Tames sarcasm |
| Theme Bias | "Favor fantasy" | More magic stories |
| Response Length | "Short replies" | Concise chats |
